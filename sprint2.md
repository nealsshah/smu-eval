# Sprint 2

**User Story:**  
As a professor, I need to import students, courses, and their groups so that peer evaluations can be assigned.

---

## OKR 2.1: Quantity (Professor)

- **Metric:** Number of courses successfully imported by professors per semester  
- **Baseline:** 0 courses imported  
- **Target:** ≥ 5 courses imported per semester (or test cycle)  
- **Unit:** Count  

---

## OKR 2.2: Content (Professors)

- **Metric:** Percentage of imported student records passing validation checks  
  (no missing fields, no duplicate IDs, valid course mapping)  
- **Baseline:** 0% validated  
- **Target:** ≥ 95% of imported records pass validation  
- **Unit:** Percentage  

---

## OKR 2.3: Automation (Student)

- **Baseline:** Students are notified via automated email of assigned peer evaluation groups  
- **Metric/Unit:** Number of groups successfully created and assigned  
- **Target:** ≥ 90% of professors can create at least one group per course within 5 minutes  
- **Unit:** Count
